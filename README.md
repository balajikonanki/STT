# Spotify Playlist to Text File (STT)

This Python script converts any Spotify playlist into a text file named after the playlist. Each line in the file contains the track name and artist.

## Features
- Fetches playlist details using the Spotify API.
- Saves the playlist tracks to a text file.
- Dynamically names the output file based on the playlist name.
- Handles special characters in playlist names.

## Requirements
- Python 3.x
- `spotipy` library
- `python-dotenv` library
